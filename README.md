# H-AAT

Analyze Windows 10 User PC Artifact by Batch Script<br>
Using Tool : RegRipper2.8 , Bootstrap, notepad++<br>
Run Win10_Aritifact_Analyze_PC.bat<br>
Script will analyze User PC or Laptop and make Report Page<br>
